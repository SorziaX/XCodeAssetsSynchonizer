<h1 align="center">XcodeAssetsSync</h1>

## Description
rename xcode asset resource files to its asset name

## Command
`assetSync [option]`

__-d --directory:__
The directory that used as root directory.

## Config File
__assetSync.json:__
`{
"path" : "."
}`

## Example
`assetSync`
`assetSync -d "."`
